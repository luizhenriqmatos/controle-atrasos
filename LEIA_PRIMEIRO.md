## 🚀 QUICK START (Resumo Rápido)

```
PASSO 1: Criar conta GitHub (se não tiver)
         https://github.com/signup

PASSO 2: Instalar Git (se não tiver)
         https://git-scm.com/download/win

PASSO 3: Criar repositório
         https://github.com/new
         Nome: "controle-atrasos"

PASSO 4: No PowerShell, na pasta do projeto:
         git config --global user.name "Seu Nome"
         git config --global user.email "seu@email.com"

PASSO 5: Fazer push
         git init
         git add .
         git commit -m "v1.0.0: Release inicial"
         git remote add origin https://github.com/SEU-USUARIO/controle-atrasos.git
         git branch -M main
         git push -u origin main

PASSO 6: Deploy no Netlify
         1. Acesse https://netlify.com
         2. "Sign up with GitHub"
         3. "Connect Git Repository"
         4. Selecione "controle-atrasos"
         5. Clique "Deploy"
         ✅ PRONTO! App ao vivo em ~2 minutos!

PASSO 7: Testar app em produção
         Clique no link do Netlify
         Login: admin123
         Teste editar dados

PASSO 8: (Opcional) Configurar senha segura
         Gere hash em: https://bcrypt-generator.com/
         Coloque em Netlify → Environment Variables

PASSO 9: Fazer backups regularmente
         Login na app
         Clique "📥 Exportar Dados"
         Guarde arquivo JSON

PASSO 10: Compartilhar URL com seu team
          https://seu-projeto.netlify.app
```
